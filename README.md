## 🏛️ Constitutional Values & Pillars

NEWGAP aligns directly with the values enshrined in the Preamble to the Constitution of India:

1. **Sovereign**: India belongs to its people — not to corporate donors, foreign lobbies, or dynastic houses. NEWGAP commits to zero dependence on corporate funding, full transparency in party finances, and citizen-first governance.
2. **Socialist**: Wealth, opportunities, and resources must be distributed justly. We stand for merit-based employment and closing the economic gap.
3. **Secular**: The state has no religion (Articles 25–28). Every religion practiced by every Indian citizen is respected equally. We never campaign in the name of faith or divide neighbors.
4. **Democratic**: Accountability between elections. We protect the right to a free press, an independent judiciary, and open democratic grievance redressal.
5. **Republic**: Leadership is earned through education, integrity, and merit — never inherited through dynastic connection or party sycophancy.
6. **Justice**: Social, economic, and political. Guaranteed grievance hearings, ensuring fairness across all public institutions.

*   **Zero Corruption**: Complete financial disclosure for all candidates and their families; refuse all corporate donations in favor of transparent crowdsourced funding.
*   **Educated Leadership**: Enforce academic qualifications for representatives (candidates need a Bachelor's; Education and PM roles require a PhD).

*   **Independent Media**: Review and cancel media licenses captured by monopoly ownerships to restore press independence.
*   **Digital-First & Education**: Implement digital grievance portals with strict timelines, open-source governance, and universal quality education for every citizen.

## 🛠️ Technical Implementation

NEWGAP is built using a modern, performant, and search-optimized stack:

*   **Framework**: [Next.js](https://nextjs.org/) (App Router).
*   **Styling**: Modern, responsive utility classes built on custom typography and CSS design tokens.
*   **ID Card Rendering**: A client-side HTML5 Canvas generator (`buildCardCanvas`) that compiles membership cards at `860x440` proportions (2x scale factor for crystal clear resolution).
*   **Google Sheets Integration**: Fully integrated with a Google Form response backend.
*   **CORS-Free Validation**: Serves a server-side route proxy (`/api/check-records`) to fetch registered candidates from Google Sheets, verifying email uniqueness and dynamically resolving duplicate Member IDs before submit.
*   **SEO Optimization**: Real-time dynamic XML sitemap (`/sitemap.xml`) and crawler controller (`/robots.txt`) combined with structured OpenGraph & Twitter Card headers.

## 🚀 Getting Started

### 1. Installation
Install the project dependencies locally:
```bash
npm install
```

### 2. Run the Development Server
Start the local server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to view the application.

### 3. Build & Production compilation
Compile the production package:
```bash
npm run build
```
